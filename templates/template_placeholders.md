# Template Placeholders

Supported placeholders:

- `{{title}}`
- `{{topic}}`
- `{{document_type}}`
- `{{audience}}`
- `{{generated_at}}`
- `{{model_name}}`
- `{{rag_mode}}`
- `{{collection}}`
- `{{thread_id}}`
- `{{body}}`

If `{{body}}` is missing, generated content is appended to the end of the
document and a warning is returned.

Microsoft Word table of contents fields must be refreshed manually by
right-clicking the TOC and selecting update field.
